About this site

This site is written in Markdown and built using the static site generator HUGO ➚. Hosted on Neocities ➚.

Designed for Firefox ➚ with Visual Studio Code ➚.

Originally built on Windows and later on the Linux Mint ➚ OS.

Open source, all code and assets are available on the Github repository ➚. If you have any suggestions of features I can add to the site or code improvements, feel free to contact me. Or if you prefer, make a pull request or open an issue on Github.
