Kat elected most awesome programs on the web

Software that I use, approve and recommend.

  • Firefox ➚ Best browser on the web if you disagree GET OUT!!!!!! FOXES > > > LIONS (specially when the lions are funded by the ceo of palantir ➚ )
  • uBlock Origin ➚ The best adblocker and annoyance-get-ridder you can get. You’re never gonna see an ad again. Free, open source, doesn’t work on Chrome (that’s a positive for me), works seamlessly with Firefox.
  • yt-dlp ➚ The most powerful video and audio downloader that exists currently. Completely free, open source and runs everywhere. Every other video downloader out there is, 9 times out of 10, a wrapper of yt-dlp. Yt-dlp is a command line application with a heap of options. If you’re not very tech savvy or just don’t wanna bother learning it, Cobalt Tools ➚ is a great alternative. CT is a yt-dlp wrapper and like all wrappers it’s options are much more limited than the base program, i’d personally recommend you pick up yt-dlp directly, there’s tons of support out there, the documentation is great, and it’s incredibly reliable, you’re never gonna have trouble when you need to download a video or song ever again.
  • Jellyfin ➚ Free, open source media library. It’s like having your own offline Netflix and you can organize it however the hell you want. It’s AWESOME!!! It also has a pretty large library of plugins, a personal favorite of mine is Youtube Metadata ➚ . It grabs the JSON file you downloaded with your video ( yt-dlp ➚ hint hint) and makes Jellyfin process the info from that file so it thinks the video is just a standard movie or episode from a show you downloaded, which means it can actually organize and read them properly. For someone who’s never used any program like this, it may seem stupid, but believe me, once you try doing anything web related in a media library like Jellyfin or Plex (JF but worse), you’ll understand why this is a lifesaver. Can you tell I’m passionate about this stuff?
  • qBittorrent ➚ Best torrent software available on the webs.
  • Obsidian ➚ Note taking app in markdown. Very complete, has an extensive online community and plugin library - anything you think you could ever think to do with a text file, Obsidian probably a way to do it. Would you believe me if I said most of the pages for this site were written there? When I was done I just dropped them into HUGO to do it’s magic. Pretty cool, huh? If you’re into writing or note taking this is a must have imo.
  • FreeFileSync ➚ Super easy to use and works the seamlessly on Linux and Windows. I use it to backup folders locally and move stuff between my phone and PC.

Resources

Resources to help build and populate your webpage.

Learn Coding

  • Bro Code ➚ Great channel for learning if you’re just getting started with HTML, CSS and JS. Do keep in mind though, I would only recommend his tutorials if you never had any contact whatsoever with these coding languages and have limited programming knowledge. Once you start to learn more, you realize he glazes over a lot of important concepts in order to keep the videos simple and understandable. Again, great for beginners, not so great if you already know some stuff. I started learning through his HTML & CSS Course ➚ .
  • MDN Web Docs ➚
  • W3Schools ➚
  • Stack Overflow ➚ The three websites you’re gonna see the most when searching up anything HTML, CSS and JS related. MDN Web Docs and W3Schools both feature documentation on all the different tags and functions ur gonna encounter on your coding journey. Mozilla’s documentation is a lot more complete, but also harder to understand, for beginners W3Schools is better. Different from the last two, Stack Overflow is a forum for asking questions, chances are whatever weird quirk you have going on with your code, someone else’s already encountered it before and asked that question there.

Tools

  • Ezgif GIF compressor ➚ Ezgif is a great tool in general for working with and making gifs, but their gif compressor is a god send, specially when you have very limited storage for your site.
  • TinyPNG ➚ My go to favorite for compressing images, supports multiple file formats and has a great compression rate.
  • CSS Gradient ➚ Create a gradient on the site and just copy the code to yours. SO helpful!
  • Visual Studio Code ➚ My text editor of choice for coding, probably the most popular one out there too. Lots of extensions to help you out. One of my favorites is the self explanatory Code Spell Checker ➚ .
  • Github ➚ Not exactly for beginners Must learn if you’re serious about web development, but if you just wanna make casual sites, you can skip it. Very powerful tool, but at the very core of it, it’s just uploading your site to a remote folder (repository) so you have a backup and nice list of changes you can revert back to if you mess up something. Github also offers free static website hosting via Github Pages ➚ .
  • deploy to neocities ➚ also not for beginners Automatically deploy your site from Github to Neocities.
  • HUGO ➚ I ABSOLUTELY DO NOT RECOMMEND THIS IF YOU ARE JUST STARTING OUT WITH WEB DEVELOPMENT Hugo is a static site generator, it’s blazing fast, super popular so there’s lots of support out there and it’s gonna help you automate so much dumb website making stuff. Pages are written in markdown ➚ so it makes writing your blog posts and whatever else a much less daunting/intimidating task. Specially when you integrate it with Obsidian ➚ . If you’re at a sort of medium level in your HTML/CSS/JS journey, I’d definitely recommend looking into SSG generators. HUGO is what makes this site possible!
